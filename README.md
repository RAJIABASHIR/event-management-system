Description: Event Management System is a full-stack web application where users can browse public events and authenticated organizers can create and manage their own events from a protected dashboard. The landing page features a modern hero section, feature highlights, spotlight events, testimonials, and stats. Logged-in users can add new events and manage (view/delete) existing ones through a clean, responsive UI that works smoothly on both desktop and mobile screens.

Technologies: Next.js (App Router), React, Firebase Authentication, Node.js, Express.js, MongoDB, JavaScript, HTML, custom CSS (responsive layout with Flexbox/Grid), React Hot Toast.
